🎬 Netflix Data Dashboard (Tableau)
📊 Project Overview

This project presents an interactive Netflix Data Dashboard created using Tableau.
The dashboard analyzes Netflix content to explore patterns in movies and TV shows, including their distribution by type, country, release year, rating, and genre.

The goal of this dashboard is to demonstrate data visualization and storytelling skills using Tableau.

📁 Dataset

The dataset used for this project is the Netflix Titles Dataset, which contains information about Netflix content such as:

Title

Type (Movie or TV Show)

Director

Cast

Country

Date Added to Netflix

Release Year

Rating

Duration

Genre (Listed In)

Description

📌 Dashboard Features

The dashboard includes multiple visualizations to analyze Netflix content:

Total Movies vs TV Shows

Content Added Over Time

Top Countries Producing Netflix Content

Distribution by Content Rating

Genre Distribution

Release Year Trends

These visualizations help users understand how Netflix content has evolved over time.

📈 Key Insights

Some insights that can be derived from the dashboard include:

Netflix has more Movies than TV Shows in its catalog.

The number of titles added to Netflix has increased significantly after 2015.

The United States produces the highest number of Netflix titles.

Certain genres such as Drama, Comedy, and International Shows dominate the platform.
